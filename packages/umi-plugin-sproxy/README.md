## umi-plugin-sproxy

安装方法:
```bash
npm i @smock/umi-plugin-sproxy
# 或
yarn add @smock/umi-plugin-sproxy
```

指定NO_PROXY环境变量可禁用拦截工具
例如:
```bash
export NO_SPROXY=true
```

拦截工具参考文档：
[https://www.npmjs.com/package/@smock/interceptor](https://www.npmjs.com/package/@smock/interceptor)
