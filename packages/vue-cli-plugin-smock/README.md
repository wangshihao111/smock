## Usage

### Installation:

```bash
npm i @smock/vue-cli-plugin-smock
```

### Modify your packages.json


```json
{
  // ...
  "scripts": {
    "smock": "vue-cli-service smock -p 6000"
  }
  // ...
}

```

Then you can run command to start the interceptor service.

```
npm run smock
```
